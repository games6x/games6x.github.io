Technical Support

Why won't a game load?

If a game isn't loading, try these steps:

  • Refresh the page (F5 or Ctrl+R)
  • Clear your browser cache and cookies
  • Try a different browser (Chrome, Firefox, Edge)
  • Check your internet connection
  • Disable ad-blockers or browser extensions that might interfere

Why is the game running slowly?

Game performance can be affected by:

  • Multiple browser tabs or programs running
  • Older computer hardware
  • Slow internet connection
  • Too many browser extensions enabled

Try closing unnecessary tabs and programs, and ensure you have a stable internet connection.

What browsers are supported?

We support all modern browsers including:

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
  • Opera

We recommend using the latest version of your browser for the best experience.

Can I play on mobile devices?

Yes! Many of our games work great on mobile devices. Simply visit our website from your smartphone or tablet's browser. Some games are optimized for touch controls.

Most games use standard controls:
